#  Movie Search App

A responsive **Movie Search App** built using **HTML, CSS, and JavaScript**. This application allows users to search for any movie and instantly view its details by fetching real-time data from a Movie API.

##  Features

*  Search movies by title
*  Fetches real-time movie data from an API
*  Displays movie poster
*  Shows IMDb rating
*  Displays release year
*  Shows genre and runtime
*  Displays movie plot/description
*  Responsive design for desktop and mobile

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Movie API (OMDb API or similar)

##  How It Works

1. Enter a movie name in the search box.
2. Click the **Search** button.
3. The app sends a request to the Movie API.
4. Movie details are fetched and displayed on the screen.
